# Mason Morrow Website

A professional website for Mason Morrow - Theatre Educator, Performer, and Playwright.

## Tech Stack

- **Nuxt 4** - Vue.js framework
- **NuxtUI** - UI component library
- **Tailwind CSS** - Utility-first CSS framework

## Getting Started

### Install Dependencies

```bash
pnpm install
```

### Development

Run the development server:

```bash
pnpm dev
```

Visit `http://localhost:3000` to see your site.

### Build

Build for production:

```bash
pnpm build
```

### Preview

Preview the production build:

```bash
pnpm preview
```

## Project Structure

- `pages/` - Application pages and routes
- `layouts/` - Layout components
- `assets/` - Styles, images, and other assets
- `public/` - Static files served as-is
- `nuxt.config.ts` - Nuxt configuration

## Pages

- **Home** (`/`) - Hero banner, intro, and feature list
- **Resume** (`/resume`) - Three resume sections (Teaching, Performance, Technical)
- **Performance** (`/performance`) - Showcase of theatrical performances with photos
- **Playwright** (`/playwright`) - Sample plays with descriptions
- **Contact** (`/contact`) - Email contact information
